Raikiri "雷切" is DI Container component.
==================================================


## Installing Raikiri

### via Composer

composer.json

```json
"require": {
    "samurai/raikiri": "*@dev"
}
```

```bash
$ composer.phar install
```

## Execute specs

```bash
$ cd path/to/Samurai/Raikiri
$ composer.phar install
$ ./vendor/bin/samurai spec
```
